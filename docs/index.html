// core primitives
Built different.
🏛
Agent Council
4 specialized AI agents deliberate every proposal in real-time. Conservative, Risk, Neutral, and Veto — each with a distinct mandate.
🔐
OWS Policy Engine
Private keys never exposed to application code. The OWS policy contract gates every transaction — 51% YES required to sign.
⚖️
Weighted Voting
Voting power is proportional to your ETH stake. Deposit more, influence more. Fully transparent on-chain.
// autonomous agents
Meet the council.
VetoAgent
Security auditor. Hard-vetoes blacklisted recipients instantly. First to vote. One NO ends the proposal.
ConservativeAgent
Financial advisor. Rejects if amount >20% of treasury or if too many transactions occurred in the last 24h.
RiskAgent
Growth investor. Always bullish — unless the proposal would drain >80% of the treasury.
NeutralAgent
Balanced analyst. Approves below 50% threshold. Recognizes repeat recipients as lower risk.

// protocol flow
Three steps. No compromises.
01 //
Propose
Any member with ETH staked can propose a withdrawal. Specify recipient, amount, and reason.
02 //
Agents Deliberate
AI agents analyze the proposal in real-time. Each votes YES or NO based on treasury health, recipient history, and risk profile.
03 //
OWS Signs
If YES ≥ 51% by voting power, the OWS policy engine authorizes the transaction. Keys never leave the wallet.
