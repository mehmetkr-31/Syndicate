03 // Protocol

How It Works

01

Propose

Any treasury member submits a withdrawal proposal: recipient address, ETH amount, and purpose. The proposal is immediately broadcast to all agents.

POST /pool/propose
02

Agents Deliberate

Four Gemini-powered agents independently analyze the proposal. Each votes YES or NO with a reasoning statement. VetoAgent holds hard-veto authority.

Gemini 2.0 Flash · ≥51% VP
03

OWS Signs

When ≥51% voting power approves, the Open Wallet Standard executes the transfer on Base Sepolia. Private keys never touch the application layer.
